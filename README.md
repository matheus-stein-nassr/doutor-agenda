# Doutor Agenda

Um sistema simples para gerenciamento de agendas médicas.

## Funcionalidades

- Cadastro de médicos e pacientes
- Agendamento de consultas
- Visualização de horários disponíveis

## Tecnologias

- Node.js
- Express
- MongoDB
- Mongoose
- React
- Bootstrap
- JavaScript

## Como usar

1. Clone o repositório
2. Instale as dependências
3. Execute o projeto
